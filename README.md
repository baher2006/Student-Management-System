# 🎓 Student Management System

A Java console-based application for managing student records using Object-Oriented Programming (OOP). The project allows users to perform basic student management operations and demonstrates the use of Java collections and file handling.

---

## ✨ Features

- ➕ Add a new student
- 📋 Display all students
- 🔍 Search for a student by ID
- ✏️ Update student information
- ❌ Delete a student
- 💾 Save student data to a text file
- 📂 Load student data automatically when the application starts

---

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling (FileWriter, File, Scanner)
- Exception Handling (try-catch)

---

## 📂 Project Structure

```
src/
├── Main.java
├── Student.java
└── StudentManager.java
```

---

## 🚀 How to Run

1. Clone or download the repository.
2. Open the project using IntelliJ IDEA (or any Java IDE).
3. Compile and run `Main.java`.
4. Use the console menu to manage students.
